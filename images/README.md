## Loose Drawing からのダウンロード手順

1. ブラウザで [https://loosedrawing.com/](https://loosedrawing.com/) を開きます。
2. サイト上部の検索ボックスに、上記テーブルの「検索キーワード」を入力して検索します。
3. 気に入ったイラストをクリックして詳細ページを開きます。
4. **「PNG」または「SVG」** のダウンロードボタンをクリックします（無料・商用利用可）。
5. ダウンロードされたファイルを、上記テーブルの **「ファイル名」欄と完全に同じ名前** にリネームします。
   - 例: ダウンロードしたファイルが `illust_123.png` であれば → `hero-illust.png` に変更

> **利用規約の確認:** Loose Drawing は商用・非商用問わず無料で利用できますが、ダウンロード時に最新の利用規約を必ずご確認ください。

## GitHub へのアップロード手順（Web インターフェース）

### 前提
- リポジトリが GitHub 上にプッシュ済みであること
- 画像ファイルが正しいファイル名にリネーム済みであること

### 手順

1. GitHub でこのリポジトリのページを開きます。
2. `images/` フォルダをクリックして移動します。
3. 右上の **「Add file」** ボタンをクリックし、**「Upload files」** を選択します。
4. 「Drag files here to add them」エリアに、リネーム済みの画像ファイルをドラッグ＆ドロップします（複数ファイル同時可）。
5. ページ下部の **「Commit changes」** セクションで、コミットメッセージを入力します。
   - 例: `Add illustration images for site sections`
6. **「Commit changes」** ボタンをクリックして完了です。

アップロード後しばらくするとサイトに反映されます（GitHub Pages を使用している場合）。

## ファイル命名ルール

- **ファイル名は上記テーブルに記載されたものを正確に使用してください。**
- スペース・大文字・日本語は使わないでください（URLエンコードの問題が起きます）。
- 拡張子は `.png` を使用してください（サイトのHTMLが `.png` を参照しています）。
- 画像が用意できていなくてもサイトは動作します。画像のない箇所には「イラスト準備中」のプレースホルダーが表示されます。
